# HupuKiller

### 0. Brief Introduction

基于 LLAMA3-8B-Chinese-Chat 基座模型的NBA知识问答系统，通过使用爬虫爬取NBA Official Website、各大论坛的NBA相关数据，利用其中15%的数据对模型使用Lora微调，剩余数据接入本地知识库用以RAG。
